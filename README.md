# Bank Customer Churn Analysis Dashboard (Power BI)

## 📊 Overview
This project focuses on analyzing bank customer churn using Power BI. The dashboard provides insights into customer behavior, churn rate, and key factors influencing customer retention.

## 📌 Key Metrics
- Total Customers: 10K
- Churn Rate: 20.4%
- Customer segmentation by gender, country, and activity status

## 🔍 Insights
- Churn trends across age groups and credit scores
- Impact of account balance on churn rate
- Customer distribution by products and geography

## 🛠 Tools Used
- Power BI
- Excel
- Data Analysis
